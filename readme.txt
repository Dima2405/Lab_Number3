Program.cs - Main, где пользователь вводит  верхнюю границу n для поиска простых чисел.

ParallelWork.cs - реализует несколько параллельных алгоритмов для поиска простых чисел.

SequentialWork.cs - который реализует последовательные алгоритмы для поиска простых чисел (Последовательный алгоритм "Решето Эратосфена" и Модифицированного последовательного алгоритма).

Calculator.cs - отвечает за выполнение различных алгоритмов поиска простых чисел и измерение их времени выполнения.

PrimeNumbers.Tests\PrimeNumbersTests.cs - Тестирование каждого метода на проверку корректности работы определенного алгоритма поиска простых чисел.
